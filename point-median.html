Ajouter un raccourcis pour le point médian sur Windows

  • Télécharger le logiciel Clavier+.
  • Dans l'application Clavier+, cliquer sur "+" pour ajouter un raccourcis, choisir "Caractère spécial", puis "183-".
  • Entrer le raccourci voulu. Pour se conformer au standard Linux, nous recommandons "AltGr + .".
